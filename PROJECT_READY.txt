BEFORE BUILDING - REQUIRED CONFIGURATION
=================================================================

1. ADD DEVELOPMENT TEAM
   - In Xcode: Select project → Signing & Capabilities
   - Add your Apple Developer team
   - Or enable "Automatically manage signing"

2. UPDATE AUTH0 CREDENTIALS
   - File: Services/AuthManager.swift
   - Lines to update:
     private let domain = "YOUR_AUTH0_DOMAIN"
     private let clientId = "YOUR_AUTH0_CLIENT_ID"

3. UPDATE API URL
   - File: Services/APIService.swift
   - Line to update:
     static let baseURL = "http://localhost:3000"

4. (OPTIONAL) CHANGE BUNDLE ID
   - Default: com.hackcanada.idle
   - Change in: Project Settings → General → Bundle Identifier
